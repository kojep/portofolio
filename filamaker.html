FilaMaker

Closed-Loop PET Pultrusion System

Purwarupa mesin pultrusi otomatis yang mengubah limbah botol PET menjadi filamen 3D printing 1.75mm. Dirancang dengan sistem kendali diameter closed-loop dan kontrol suhu PID ganda untuk memastikan stabilitas dimensi serta mencegah degradasi polimer.

ESP32-S3 PID CONTROL CUSTOM PCB C/C++

// 01. THE_CHALLENGE

Problem Statement & Keterbatasan Teknis

Inkonsistensi Diameter

Limbah botol PET memiliki ketebalan bervariasi. Sistem pultrusi DIY (open-loop) menghasilkan diameter filamen yang tidak stabil, memicu masalah ekstrusi pada 3D printer.

Degradasi Termomekanis

PET bersifat higroskopis. Pemanasan plastik lembab memicu degradasi hidrolitik, membuat filamen rapuh dan memiliki layer adhesion yang buruk.

Resiko Mikroplastik

Gesekan mekanis pita PET melewati nozzle pemanas menghasilkan partikel mikroplastik yang berisiko terhirup oleh operator di ruangan tertutup.

// 02. THE_SOLUTION

Kebaruan Inovasi FilaMaker

  • Closed-Loop Control: Sistem pengendalian diameter adaptif menggunakan sensor Hall effect dan algoritma PID.
  • Thermal PID & Dehumidification: Kontrol suhu ekstrusi presisi tinggi dipadukan dengan tahap pengeringan untuk mencegah degradasi hidrolitik polimer.
  • Purification Module: Integrasi sistem filtrasi berpori untuk menyaring partikel mikroplastik demi standar Keselamatan dan Kesehatan Kerja (K3).

// 03. ENGINEERING_DESIGN

System Architecture & Mechanics

Sistem Elektronika & Kendali

Diotaki oleh mikrokontroler ESP32-S3, FilaMaker memproses data dari Termistor NTC 100k dan Sensor Hall Effect melalui modul ADS1115. Sinyal output kemudian mengendalikan elemen pemanas (via MOSFET) dan motor penarik (via driver TMC2209).

PID CONTROL ALGORITHM

u(t) = Kp e(t) + Ki ∫ e(t)dt + Kd de(t)⁄dt

Algoritma ini diterapkan secara ganda: satu untuk menstabilkan suhu (Thermal PID) dan satu lagi untuk menyesuaikan kecepatan putaran Planetary Gear secara adaptif agar diameter filamen tetap konsisten di angka 1.75mm.

Desain Mekanik (3 Zona)
  • Zona Input & Pemanas: Dudukan spool, corong pemandu, dan hotend dengan nozzle 1.7mm.
  • Zona Purifikasi & Sensor: Kipas pendingin terarah, sensor pembaca diameter real-time, dan media filtrasi penangkap mikroplastik.
  • Zona Penarik: Spool penggulung yang terintegrasi dengan NEMA 17 dan Planetary Gear (rasio 6:1) untuk torsi penarikan maksimal dengan desain ringkas.

// 04. FUTURE_ROADMAP

Tahap Pengujian & Validasi

Proyek ini sedang dipersiapkan untuk maju dalam Program Kreativitas Mahasiswa (PKM-KC) 2026. Tahap akhir dari perakitan akan melibatkan validasi ketat dengan parameter berikut:

1. Presisi Dimensional

Target stabilitas diameter di angka 1.75mm dengan batas toleransi maksimal ±0.05 mm.

2. Uji Purifikasi

Observasi kuantitatif terhadap akumulasi residu mikroplastik pada modul filter pasca-produksi.

3. Fungsionalitas Cetak

Mencetak objek uji (Benchy/Calibration Cube) untuk menilai konsistensi aliran ekstrusi dan ikatan antar lapisan (Layer Adhesion).

4. Validasi Mekanik

Pengujian kekuatan tarik sampel menggunakan Universal Testing Machine (UTM) sebagai komparasi dengan filamen komersial.
