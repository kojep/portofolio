S.T.E.L.L.A

System That Explain Literally Like AI

Pengembangan voice assistant fisik cerdas yang mengintegrasikan pemrosesan bahasa alami mutakhir dengan rekayasa perangkat keras. S.T.E.L.L.A bukan sekadar chatbot, melainkan entitas asisten pintar dengan persona yang mampu berinteraksi secara vokal dengan intonasi selayaknya manusia.

GEMINI API ELEVENLABS API PYTHON HARDWARE DEV

// 01. SYSTEM_ARCHITECTURE

Pipeline Komputasi Kognitif

1
Input & STT

Menangkap sinyal suara analog pengguna melalui modul mikrofon terintegrasi, kemudian mengonversinya menjadi teks digital menggunakan engine Speech-to-Text dengan latensi rendah.

2
Gemini API (The Brain)

Teks masukan dikirim ke Gemini API. Prompt dirancang khusus dengan system instructions agar S.T.E.L.L.A memiliki persona teknis, cerdas, dan merespons secara ringkas dan relevan layaknya asisten lab.

3
ElevenLabs (TTS)

Teks balasan dari Gemini diproses oleh ElevenLabs API. Sintesis suara (*Text-to-Speech*) ini menggunakan model suara premium untuk menghasilkan intonasi dan emosi yang sangat natural melalui speaker.

// 02. PHYSICAL_FORM

Integrasi Ekosistem Hardware

S.T.E.L.L.A bukan sekadar skrip Python di layar monitor, melainkan wujud fisik yang membumi. Sebagai hardware maker, saya merancang perangkat ini agar dapat berinteraksi langsung di meja kerja.

  • Custom 3D Printed Chassis: Casing didesain via CAD dan dicetak 3D secara mandiri dengan mempertimbangkan akustik ruang speaker dan ventilasi udara mikroprosesor.
  • SBC / Microcontroller Brain: Menggunakan komputasi embedded system untuk memproses script I/O, merekam audio, dan menjembatani panggilan API secara real-time via koneksi Wi-Fi.
  • Visual Feedback: Integrasi lampu indikator LED cerdas (contoh: Neopixel) yang bereaksi sesuai *state* sistem (Mendengarkan, Memproses, Berbicara).
